# MyQuant

S&P 500 多因子量化选股系统。基于 LightGBM + 多策略框架，经历 4 轮迭代，从 6 因子单模型发展到 33 因子多策略+风控体系。

## 项目概览

| 项目 | 详情 |
|------|------|
| 标的池 | S&P 500 中 100 只股票 |
| 数据范围 | 2015-01 ~ 2025-12（~11 年） |
| 样本外评估期 | 2018-01 ~ 2025-12（~7.9 年） |
| 标签 | 未来 5 日超额收益（个股 - SPY） |
| 模型 | LightGBM + Ridge（baseline 对照） |
| 最终回测夏普 | **1.16**（扣除全部成本后） |
| 最终年化收益 | **18.37%**（SPY 同期 15.16%） |

## 四轮迭代

### V1 → V2 → V3 → V4 演进路径

```
V1: 6 因子 + LightGBM → 夏普 0.70
 ↓ 因子从 6 → 25，信号质量大幅提升
V2: 25 因子 + LightGBM → 夏普 2.04
 ↓ 加入降换手控制，牺牲收益换取可执行性
V3: 25 因子 + 降换手 → 夏普 1.01
 ↓ 加入 VIX/利率/美元跨资产因子
V4: 33 因子 + 降换手 → 夏普 2.78
 ↓ 加入多策略框架 + 三层风控
最终: 多策略 + 风控 → 成本后夏普 1.16, 回撤 -24.74%
```

## 因子体系（V4: 33 个因子）

### 股票因子（25 个）

| 类别 | 因子 |
|------|------|
| **动量/反转 (6)** | return_1d, momentum_5d/20d/60d, return_5d_reversal, high_52w_dist |

| **技术指标 (5)** | rsi_14, ma_deviation_20d, macd_signal, bollinger_pos, atr_14_pct |
| **微观结构 (3)** | close_position, gap_open, amihud_illiq_20d |
| **高阶统计 (3)** | return_skew_20d, return_kurt_20d, downside_vol_20d |

| 类别 | 因子 |
|------|------|
| **VIX (3)** | vix_level, vix_change_5d, vix_term_structure |

## 多策略框架

| 策略 | 逻辑 | 夏普 | 与策略A相关性 |
|------|------|------|-------------|
| **A: ML 多因子** | LightGBM 33 因子预测 | 2.78 | — |
| **B: 短期反转** | 过去 5 日跌最多 → 买入 | 1.21 | 0.27 |
| **C: 动量趋势** | 过去 60 日涨最多 → 买入 | 1.20 | 0.27 |
| **D: 低波动** | 波动率最低 → 买入 | -0.20 | -0.11 |

合并方式：按历史夏普加权（策略 D 权重接近 0）。

## 风控模块

| 层级 | 机制 | 参数 |
|------|------|------|
| **第一层** | 目标波动率缩放 | 目标 12%，实际波动高时自动降仓 |
| **第二层** | 最大回撤熔断 | 回撤 >15% → 仓位减半，>25% → 仓位降至 20% |
| **第三层** | 行业集中度 | 单行业 ≤ 30% |

## 快速开始

### 环境安装

```bash
pip install -r requirements.txt
pip install lightgbm
```

### 运行全流水线

```bash
# 1. 下载数据
python data/get_sp500_tickers.py
python data/download_sp500_100.py
python data/download_spy.py

# 2. 数据清洗
python data/merge_raw_data.py
python data/calender_align.py

# 3. 因子工程
python feature/build_factors_v3.py

# 4. 策略训练（V4: 33因子 + 降换手）
python strategy/strategy_v4.py

# 5. 多策略合并
python strategy/multi_strategy.py

# 6. 回测（含风控）
python backtest/backtest_v2.py

# 7. 模拟实盘成本分析
python backtest/simulate_live.py
```

## 技术栈

| 组件 | 技术 |
|------|------|
| 语言 | Python 3.11 |

| 机器学习 | LightGBM, scikit-learn (Ridge) |
| 数据源 | yfinance (免费) |
