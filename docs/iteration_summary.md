# MyQuant 量化项目迭代总结

## 项目概览

| 项目 | 详情 |
|------|------|
| 标的池 | S&P 500 中 100 只股票 |
| 数据范围 | 2015-01-02 ~ 2025-12-31（~11年） |
| 实际评估期 | 2018-01 ~ 2025-12（~7.9年样本外） |
| 标签 | 未来 5 日超额收益（个股 - SPY） |
| 初始资金 | $1,000,000 |

## 四次迭代对比总览

### 因子与模型演进

| 版本 | 因子数 | 因子类别 | 模型 | 组合构建 | 调仓频率 |
|------|--------|---------|------|---------|---------|
| **V1** | 6 | 动量、波动率、量价、技术指标 | LightGBM + Ridge | Top 20 等权做多 | 每5天 |
| **V2** | 25→21(去相关) | +反转、微观结构、高阶统计 | LightGBM + Ridge | Top 20 等权做多 | 每5天 |
| **V3** | 21 (同V2) | 同V2 | LightGBM + Ridge | Top 20 + 降换手控制 | 每10天 |
| **V4** | 33→29(去相关) | +VIX、利率、美元指数跨资产因子 | LightGBM + Ridge | Top 20 + 降换手 + 多策略 + 风控 | 每10天 |

### 各版本因子清单

#### V1（6 个基础因子）
```
动量:     return_1d, momentum_20d
波动率:   volatility_20d
量价:     volume_ratio_20d
技术指标: rsi_14, ma_deviation_20d
```

#### V2（25 个 → 实际使用 21 个）
```
动量/反转(6):   return_1d, momentum_5d, momentum_20d, momentum_60d, return_5d_reversal, high_52w_dist
波动率(4):      volatility_20d, volatility_60d, vol_ratio_20_60, intraday_range
量价(4):        volume_ratio_20d, volume_ratio_5d, volume_std_20d, price_volume_corr_20d
技术指标(5):    rsi_14, ma_deviation_20d, macd_signal, bollinger_pos, atr_14_pct
微观结构(3):    close_position, gap_open, amihud_illiq_20d
高阶统计(3):    return_skew_20d, return_kurt_20d, downside_vol_20d

去掉高相关(4):  return_5d_reversal(r=-1.0), volatility_60d(r=0.83), volume_ratio_5d(r=0.80), bollinger_pos(r=0.94)
```

#### V3（同 V2，仅改组合构建层）
```
因子不变，新增降换手控制：
  - 持仓惩罚 (holding bonus = +0.2%)
  - 缓冲带 (买入Top20, 卖出阈值Top30)
  - 调仓频率 5天 → 10天
```

#### V4（新增 8 个跨资产因子）
```
VIX(3):   vix_level, vix_change_5d, vix_term_structure
利率(3):  tnx_level, tnx_change_20d, tnx_momentum_60d
美元(2):  dxy_change_20d, dxy_momentum_60d
```

## 迭代路径总结

```
V1: 6因子 + LightGBM → 夏普0.70
 ↓ 因子从6→25，信号质量大幅提升
V2: 25因子 + LightGBM → 夏普2.04
 ↓ 加入降换手控制，牺牲收益换取可执行性
V3: 25因子 + 降换手 → 夏普1.01
 ↓ 加入VIX/利率/美元跨资产因子，信息增量显著
V4: 33因子 + 降换手 → 夏普2.78
 ↓ 加入多策略框架(4策略) + 三层风控
最终: 多策略 + 风控 → 成本后夏普1.16, 最大回撤-24.74%
```

### 每次迭代的核心贡献

| 迭代 | 核心改动 | 最大贡献 |
|------|---------|---------|
| V1→V2 | 因子 6→25 | 夏普 0.70→2.04，因子数量是最大驱动力 |
| V2→V3 | 降换手控制 | 换手率降到12%，为回测做准备 |
| V3→V4 | 跨资产因子 | 夏普 1.01→2.78，宏观因子信息增量巨大 |
| V4→最终 | 多策略+风控 | 回撤 -39%→-25%，波动率 25%→16%，夏普突破1.0 |

## V4 可优化方向

### 按投入产出比排序

| 优先级 | 方向 | 具体做法 | 预期效果 | 难度 |
|--------|------|---------|---------|------|
| **1** | **模型 Ensemble** | LightGBM + XGBoost + Ridge 三模型融合 | 夏普提升 0.1-0.3，预测更稳定 | 低 |
| **2** | **基本面因子** | PE、PB、ROE、盈利增速（yfinance可取） | 与价量因子低相关，信息增量大 | 中 |
| **3** | **动态策略权重** | 多策略权重从固定→根据近期表现自适应 | 回撤进一步降低 | 中 |
| **4** | **淘汰策略D** | 低波动策略夏普-0.20，拖累整体 | 去掉负贡献 | 低 |
| **5** | **目标波动率调参** | 从12%调到10%或8% | 回撤从-25%→-15~18% | 低 |
| **6** | **扩大标的池** | 100只→300-500只 S&P 500 | 更多选股空间，分散度更好 | 中 |
| **7** | **多周期融合** | 同时用5日/10日/20日标签训练多模型 | 捕捉不同频率的信号 | 中 |
| **8** | **加入 VaR/CVaR 风控** | 基于尾部风险动态调仓 | 极端行情保护更好 | 中 |
| **9** | **情绪因子（NLP）** | 新闻/社交媒体情绪分析 | 另类数据增量 | 高 |
| **10** | **交易执行优化** | TWAP/VWAP 模拟，更真实的滑点模型 | 回测更接近实盘 | 高 |

### 当前最大瓶颈分析

| 问题 | 证据 | 解决方案 |
|------|------|---------|
| **回撤仍偏大** (-24.74%) | 2020疫情期间风控不够激进 | 目标波动率从12%→8%，或加VaR限制 |
| **熊市选股失效** (2022胜率46%) | 纯价量因子在趋势反转时滞后 | 加基本面因子（盈利质量在熊市更有效） |
| **策略D是负贡献** | 低波动策略夏普-0.20 | 直接淘汰或替换为质量因子策略 |
| **超额收益不够稳定** (IR=0.33) | 部分年份跑输SPY | 扩大标的池 + 多周期融合 |
| **成本偏高** (年化4.78%) | 多策略合并后持仓变化频繁 | 统一调仓日，减少策略间冲突 |
